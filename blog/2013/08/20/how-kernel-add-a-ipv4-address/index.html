How Kernel Add a IPv4 Address

Aug 20th, 2013 4:31 pm

summary

There is a struct in_device __rcu *ip_ptr under every net_device. which stores all the IPv4 related information for this device.

Each IPv4 address is stored a struct in_ifaddr, all the IPv4 address of a net device are stored into a list(netdev->ip_ptr->ifa_list).
